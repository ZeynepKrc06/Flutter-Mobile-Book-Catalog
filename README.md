Kitap Dünyası - Katalog Uygulaması
Bu proje, Flutter ile geliştirilmiş bir kitap kataloğu uygulamasıdır. Kullanıcıların kitaplar arasında arama yapmasını, kitap detaylarını incelemesini ve bir sepet yönetimi deneyimi yaşamasını sağlar.


Proje Hakkında
Uygulama, üç ana ekrandan oluşmaktadır:
Keşfet Sayfası: Kullanıcıyı karşılayan, arama çubuğu ve kitap listesini içeren ana ekran.
Detay Sayfası: Seçilen kitabın özelliklerini gösteren etkileşimli alan.
Sepetim: Eklenen ürünlerin listelendiği, ürün silme ve ödeme seçeneklerini içeren bölüm.


Kullanılan Teknolojiler ve Sürüm Bilgisi
Flutter Sürümü: 3.41.9
Dart Sürümü: 3.11.5
Tasarım Dili: Material 3
Programlama Dili: Dart
Widget Yapısı: Material 3 tasarımı, StatefulWidget ve StatelessWidget mimarisi kullanılmıştır.


Çalıştırma Adımları
Depoyu Klonlayın:
```bash
git clone https://github.com/ZeynepKrc06/flutter-kitap-katalog-uygulamasi.git
```

Proje Dizinine Gidin:
```bash
cd mini_katalog
```

Bağımlılıkları Yükleyin:
```bash
flutter pub get
```

Uygulamayı Başlatın:

Emülatörünüzün veya cihazınızın bağlı olduğundan emin olun.
```bash
flutter run
```
